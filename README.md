# Pinces Acoustiques (Acoustic Tweezers)

Ce projet implémente un système de manipulation de particules sans contact utilisant les forces de rayonnement acoustique. L'objectif est de piéger et de déplacer des micro-objets (cellules, billes de polystyrène) en modulant des champs ultrasonores.
Inclus : simulation des champs de pression, contrôle des phases des transducteurs et scripts d'analyse de trajectoire.
